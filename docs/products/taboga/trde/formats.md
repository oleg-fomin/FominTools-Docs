# Export Formats

## 📊 Supported Formats

### 📑 Excel Format
- XLSX and XLS support
- Custom formatting
- Multiple sheets
- Formula support

### 🌐 HTML Format
- Responsive layouts
- CSS styling
- Interactive tables
- Mobile friendly

### 📄 PDF Format
- Vector graphics
- Font embedding
- Page numbering
- TOC generation

### 📋 XML Format
- Custom schemas
- XSLT support
- Data validation
- UTF-8 encoding

## 🎨 Formatting Options

### 📐 Layout Control
- Margins and padding
- Column widths
- Page breaks
- Headers/Footers

### 🎯 Data Types
- Text alignment
- Number formats
- Date/Time formats
- Currency options
